# My Blog

一个全栈个人博客系统。

## 技术栈

- 前端: Next.js 14 + React 18 + TypeScript + Tailwind CSS + shadcn/ui
- 后端: Go 1.26 + Gin + PostgreSQL (pgx + sqlc)
- 部署: systemd + Nginx
